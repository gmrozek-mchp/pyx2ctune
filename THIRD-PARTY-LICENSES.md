## Firmware / Reference Content (`motorbench/`)

The `motorbench/` directory contains Microchip MCAF firmware generated by
motorBench / MCC. This content is **not distributed** with pyx2ctune — it exists
locally for reference (reading `parameters.json`, YAML configs, and header
files). It is never packaged, bundled, or uploaded.

| Component | License | File |
|---|---|---|
| MCAF / motorBench generated code | **Microchip Proprietary** | `motorbench/.../motorBench/LICENSE.txt` |
| X2Cscope C library | **BSD-3-Clause** (LCM / Microchip) | `motorbench/.../library/x2cscope/X2Cscope.h` |

The Microchip proprietary license restricts redistribution and requires use
solely on or with Microchip devices. The full text is in `LICENSE.txt` within
the motorBench tree.

The X2Cscope library carries a dual-copyright BSD-3-Clause license:
- Copyright (c) 2013, Linz Center of Mechatronics GmbH (LCM)
- Copyright (c) 2016, Microchip Technology Inc.

## Licensing Considerations

### PyQt5 Licensing Considerations

PyQt5 is licensed under GPL-3.0, which is a strong copyleft license. It is a
**transitive** dependency — pyx2cscope declares it as a requirement, and it gets
installed automatically.

**Implications:**

- **Library/notebook use:** When pyx2ctune is used as a library (imported in
  scripts or Jupyter notebooks), PyQt5 may be installed but is not directly
  invoked by pyx2ctune itself. pyx2ctune uses matplotlib for plotting, not PyQt5
  directly.

- **PyInstaller-bundled GUI:** The standalone GUI bundles all dependencies into a
  single executable. Distributing a bundle that includes PyQt5 requires
  compliance with GPL-3.0 terms (source availability for the complete
  application) **unless** a commercial PyQt5 license is obtained from Riverbank
  Computing.

- **Recommendation:** If distributing pre-built binaries, either:
  1. Obtain a commercial PyQt5 license, or
  2. Ensure GPL-3.0 compliance (offer complete corresponding source for the
     bundled application), or
  3. Replace the PyQt5 dependency chain (this would require changes in
     pyx2cscope upstream).

### Microchip Proprietary Dependencies

`pyx2cscope` and `mchplnet` are published on PyPI under a proprietary license by
Microchip Technology Inc. Users install them directly from PyPI. Redistribution
of these packages outside of PyPI requires Microchip's authorization.

The pre-built GUI executables bundle these proprietary packages. Ensure that
distribution of bundles containing `pyx2cscope` and `mchplnet` is authorized
under Microchip's terms.

### PyInstaller GPL-2.0 Exception

PyInstaller itself is GPL-2.0, but its bootloader carries a special exception
that allows the bundled application to use any license. The exception explicitly
states that linking or bundling with PyInstaller does not cause the resulting
executable to be covered by the GPL. PyInstaller is a dev-only dependency and is
not distributed with pyx2ctune.
